GET

Get Clients

  • ' /clients ' to get all clients
  • ' /clients/client/:clientID ' to get 1 client by id

Get Stock

  • ' /clients/stocks ' to get all stocks
  • ' /clients/client/:clientID/stock ' to get 1 stock from client by id

Get Stock

  • ' /clients/stocks ' to get all stocks
  • ' /clients/client/:clientID/stock ' to get 1 stock from client by id

Get Fishes

  • ' /fish ' to get all fishes

POST

Post Clients

  • ' /signup ' to create client

    • => provide a body of:{ "email": "email", "password": "password" }

  • ' /login ' to sign in

    • => provide a body of:{ "email": "email", "password": "password" }

Post Stock

  • ' /clients/stocks ' to create stock for client

    • => provide a body of:{ "client_id": "client_id", "fish_id": "fish_id", "quantity": "quantity" }

Post Fish

  • ' /fish ' to send fish to database

    • => provide a body of:{ "fish_id": "fish_id", "common_name": "common_name", "scientific_name": "scientific_name", "species_group": "species_group", "image_link": "image_link" }

UPDATE

Update Stock

  • ' /clients/client/:clientID/stock ' to update fish quantity by client id

DELETE

Delete Stock

  • ' /clients/client/:clientID/stock ' to delete fish stock by client id
